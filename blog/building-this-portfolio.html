How I Built This Portfolio

May 20, 2026 · 5 min read

Welcome to the first blog post on this site! I wanted to share the story behind building this portfolio — from the initial concept to the final code.

The Vision

I wanted a portfolio that didn't just list my skills but told a story. The lotus flower became the central metaphor — blooming from muddy waters, representing growth through challenges.

Every design decision from the color palette to the animations was made with intention. The dark background with pink and purple gradients creates a serene, focused atmosphere.

Tech Stack

This site is built with vanilla HTML, CSS, and JavaScript — no frameworks needed. Here's what makes it tick:

  • Canvas particles — A custom particle system creates the floating lotus petals
  • Glassmorphism cards — Frosted glass effect using backdrop-filter: blur()
  • GitHub API — Projects are loaded dynamically from my public repos
  • Dark/Light mode — CSS custom properties + localStorage
  • Carousels — Button-driven sliders for skills and projects

Design Details

The lotus pink #f0a1c7 and deep purple #c471ed create a harmonious gradient palette. Gold accents provide contrast for important elements. Each section fills the viewport for a clean, focused browsing experience.

Building this was a labor of love. I hope you enjoy exploring it as much as I enjoyed creating it!
